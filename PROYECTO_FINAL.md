## 🔧 Detalles Técnicos

### Arquitectura Servidor-Cliente

```
Navegador               Servidor HTTP           Intérprete
   │                         │                      │
   │──────(código JSON)──────>│                      │
   │                         │──(ejecutar código)──>│
   │                         │                      │
   │                         │<─(resultado texto)──│
   │<────(respuesta JSON)────│                      │
   │                         │                      │
```

### Flujo de Ejecución

1. Usuario escribe código en la IDE
2. Presiona Ctrl+Enter
3. JavaScript envía POST con código
4. Servidor HTTP recibe solicitud
5. Servidor ejecuta código con Intérprete
6. Captura salida en buffer
7. Responde con JSON
8. JavaScript muestra resultado
