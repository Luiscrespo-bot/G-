# 🚀 IDE Web - Guía Rápida

## En 3 Pasos

### 1️⃣ Compilar

```bash
# Windows
.\compilar_ide.bat

# Linux/macOS
./compilar_ide.sh
```

### 2️⃣ Ejecutar

```bash
# Windows
lenguaje_ide.exe

# Linux/macOS
./lenguaje_ide
```

Deberías ver:
```
✅ Servidor iniciado en: http://127.0.0.1:8080
```

### 3️⃣ Abrir en Navegador

```
http://127.0.0.1:8080
```

¡Listo! 🎉

## Escribir Código

```
var x = 10;
escribir("Valor:", x);
```

Presiona **Ctrl+Enter** o haz clic en **▶ Ejecutar**

## Cargar Ejemplos

Usa el desplegable "Cargar Ejemplo..." para:
- Hola Mundo
- Operaciones
- Condicionales
- Bucles
- Funciones

## Atajos

- **Ctrl+Enter** - Ejecutar
- **Tab** - Indentar

## Puerto Personalizado

```bash
lenguaje_ide 3000
```

Luego abre: `http://127.0.0.1:3000`
